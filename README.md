# Employee Management
With this application we can maintain the data of employees in an organization
We can add emplyees data,update data,delete data or view all the details of employees

### Tech Stack:

- Java
- Maven
- Spring Boot 
- Swagger open doc
- Docker 
- Prometheus
- Grafana
- Junit
- Postman
